# Netflix-PowerBI-Dashboard
This project is a Power BI dashboard created to analyze and visualize Netflix data. It provides key insights into movies and TV shows, genres, ratings, release years, country-wise content distribution, and trends using interactive visualizations.
The goal of this project is to demonstrate data analytics, visualization skills, and business intelligence reporting using Microsoft Power BI.
